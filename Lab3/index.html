(i) What are main benefits of using CSS?

  • Ease to control and modify by CSS;
  • Web sites will be more maintainable;
  • More accessible;
  • Ease to adopt different mediums;
  • Web pages will be quicker to download, if the web sites built using a centralized set of CSS files for all presentation.

(ii) What are different parts of a CSS style rule?

A rule consists of a selector, followed by a series of property and value pairs.

(iii) What is an element selector? Give an example.

An element selector uses the HTML element name.
e.g. div{}

(iv) What is a class selector? Give an example.

A class selector can select a series of HTML elements which are labeled with the same class attribute value.
e.g. .first{}

(v) What is an id selector? Give an example.

An id selector can select an HTML element which has been labeled with an id attribut.
e.g. #latestComment{}

(vi) What are contextual selectors in CSS? Give an example.

A contextual selector allows you to select elements based on their ancestors, descendants, or siblings.
e.g. div p{}

(vii) What is an attribute selector? Give an example.

An attribute selector can select HTML elements by either the presence of an element attribute or by the value of an attribute
e.g. [title]{}

(viii) What is a pseudo selector? Give an example.

A pseudo selector is a way to select something that does not exist explicityly as an element in the HTML document tree but which is still a recognizable selectable object.
e.g. a:hover{}

(ix) What are the three cascade principles used by browsers when style rules conflict?

  • inheritance
  • spcificity
  • location
