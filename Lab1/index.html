Q10:Describe in detail the process of DNS address resolution? (3 marks)

A10:The process of DNS address resolution
  1. Input a domain name in web browser, DNS Resolver check it's in browser's cache or not.
  2. If domain name in browser's cache, return IP address to browser and then browser requrests pages.
  3. If domain name is not in browser's cache, resolver makes request for IP address to ISP's DNS Server.
  4. If ISP's DNS Server found domain name in it's DNS cache, return IP address to DNS Resolver.
  5. If the primary DNS server doesn't have the domain name in its DNS cache, it sends out to the root name server.
  6. Root name server returns IP of name server for requested TLD.
  7. The primary DNS server request IP of name server for the domain name.
  8. gTLD server will return IP address of DNS server for this domain name.
  9. The primary DNS server request for IP address for this domain name to DNS server.
  10. DNS serer return IP address of web server to Primary DNS server, and then it return to DNS resolver.
  11. Then, DNS resolver return IP address the domain name to web browser.
  12. Web browser request page to Web server based on IP address.
  13. Web server returns requested page to web browser.
